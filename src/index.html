Amazon S3 Buckets

Amazon Simple Storage Service (S3) is a scalable, high-speed, web-based cloud storage service designed for online backup and archiving of data and applications. S3 allows you to store and retrieve any amount of data at any time, from anywhere on the web.

What is a Bucket?

A bucket is a container in Amazon S3 that holds your data objects (files). Every object is stored in a bucket, and each bucket has a unique name across all of AWS.

Key Features of S3 Buckets

  • Scalability: Automatically scales to handle large amounts of data.
  • Durability: Designed for 99.999999999% durability of objects over a given year.
  • Security: Supports encryption and fine-grained access control using IAM policies.
  • Versioning: Keep multiple versions of an object in the same bucket.
  • Lifecycle Policies: Automatically archive or delete objects based on rules.

Example of Creating a Bucket (AWS CLI)

You can create a bucket using the AWS Command Line Interface (CLI) like this: 

aws s3 mb s3://my-example-bucket

Use Cases

  • Website hosting and content delivery.
  • Backup and restore applications.
  • Big data analytics.
  • Storing media files like images, videos, and documents.
